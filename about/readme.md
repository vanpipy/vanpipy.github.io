## 项目经历

### 巽风数字世界智能NPC - 巽风科技

* 场景：原生游戏程序中的NPC以固定脚本交互为主，难以覆盖丰富的用户提问与多轮对话，运营侧也缺少可配置的角色话术能力
* 目标：在维持游戏主流程的前提下，为NPC增加基于大模型的智能问答能力，并支持提示词配置与智能体快速创建，以提升运营配置效率与交互体验
* 行动：
    - 可逆设计：采用"防腐层"架构，将LLM能力封装为独立服务层，通过特性开关控制调用。保留原有逻辑作为降级兜底，确保风险可控、可快速回退
    - 技术决策：在"微调"与"Prompt+RAG"之间选择后者，权衡依据：a. 维护成本（高昂 vs 平价）、b. 可观测性（提示词可审计可追溯）、c. 运营自助程度（无需额外协助），决策依据文档化可追溯
    - 资产沉淀：识别出Prompt模板管理、对话上下文管理、Token计费等通用模块，按"三次原则"抽取为团队公共能力，支撑后续同类需求复用
* 结果：
    - 实现NPC多轮智能问答，运营可通过配置提示词快速调整角色设定与回答风格
    - 话术迭代周期从"2周发版"缩短至"5分钟配置生效"，调试成本降低约90%
    - 沉淀的LLM组件库后续支撑公司多个产品线同类需求复用
